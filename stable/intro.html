TorXakis¶

TorXakis is a tool for Model Based Testing.

It is licensed under the BSD3 license.

TorXakis is written in Haskell and uses `stack <https://www.haskellstack.org>`__ as build tool. In addition TorXakis needs an SMT solver, such as cvc4 or Z3. The SMT Solver needs to support SMTLIB version 2.5, Algebraic Data Types (as proposed for version 2.6), and Strings. The SMT Solvers are assumed to be located on the PATH. For development acceptance tests javac is needed.

The Haddock documentation is also available here.
